# Sales-Forecasting-Time-Series-Analysis
Sales Analysis Dashboard (Power BI) Built an interactive Power BI dashboard to analyze historical sales data across different time periods. Used time-based visuals to observe trends and seasonal patterns, added KPIs and slicers for dynamic filtering, and generated insights to support basic sales analysis and planning.


Overview

This project uses Power BI to analyze historical sales data and understand sales trends over time. The dashboard provides an interactive way to explore performance across different time periods.

Tools Used

Power BI

Historical sales data

Features

Interactive dashboard with KPIs

Time-based analysis of sales trends

Slicers for dynamic filtering

Clear visual representation of sales performance

Purpose

To gain insights into sales behavior over time and support basic analysis for planning and decision-making.

Conclusion

This project demonstrates practical use of Power BI for building interactive sales dashboards and performing time-based analysis.
